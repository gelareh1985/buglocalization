Creation Time 2020-02-11T06:59:02Z
create a record with the below code public interface Shape{} record Record1() extends Shape{} The above code gives an error on Record1 as extends cannot be used for a record, but the AST is not created for Record1. In a similar case for Enum public interface Shape{} enum Enum1 extends Shape{} The above code gives an error on Enum1 as extends cannot be used for enum, but the AST is created Enum1. Record1 also should follow the way Enum1 AST is recovered and constructed.
