Creation Time 2020-04-17T09:32:20Z
A testcase has just been added via bug 562242 codified in Java14ElementProcessor#testRecords9(). In summary, a record component is annotation with the following annotation: @Retention(RetentionPolicy.RUNTIME) @Target({ElementType.RECORD_COMPONENT,ElementType.METHOD}) @interface MyAnn { int value() default 1; } public record R3(@MyAnn(value=5) Character c){} Since METHOD is also an applicable target, the accessor method corresponding to the record component "c" should also get the same annotations.
Creation Time 2020-04-17T09:47:15Z
Okay, I see that we are setting the annotations on the SyntheticMethodBinding (which is the accessor) but only as type annotations. We should also check for the target and if found to include METHOD, should also be set on the method binding itself.
