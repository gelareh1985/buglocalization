Given: import java.lang.annotation.ElementType; import java.lang.annotation.Target; @Target({ElementType.PARAMETER}) @interface MyAnnot {} public record Point(@MyAnnot() int i, int j) {} error"The annotation @MyAnnot is disallowed for this location" is flagged. However, JLS 14, Section 9.7.4 has the addition: a record component but T is not applicable to record component declarations, field declarations, method declarations, formal and exception parameter declarations, or type contexts. ecj rightfully doesn't flag error with targets of record comp decls, fields. Additionally, check for the rest on the applicability as well.
