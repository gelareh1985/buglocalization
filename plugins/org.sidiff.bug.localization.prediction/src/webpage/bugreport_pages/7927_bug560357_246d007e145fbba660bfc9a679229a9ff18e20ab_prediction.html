Subword matching was set to disabled due to its inclusion close to the end of 2020-03 M3. I think we should enable it by default for the start of 2020-06 M1. The feature seems to work well and is in part based on substring completion, but we stand a better chance of discovering any bugs if enabled by default early in the cycle.
