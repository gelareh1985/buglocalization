Creation Time 2018-10-25T19:05:39Z
Java 8 added new Javadoc tags `@apiNote`, `@implSpec`, and `@implNote`. https://bugs.openjdk.java.net/browse/JDK-8068562 The Java editor autocomplete features recognizes other tags (e.g. `@see`) but not these. These are not new tags; they have been available for years and are now standard. NetBeans (https://netbeans.org/bugzilla/show_bug.cgi?id=249793) and many other projects have been adding support. They are even discussed and recommended in _Effective Java, Third Edition_. https://dzone.com/articles/whats-new-in-effective-javas-third-edition Bug 422073 is related but is not a duplicate. See more discussion at https://blog.codefx.org/java/new-javadoc-tags/ .
