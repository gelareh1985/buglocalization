Creation Time 2020-11-11T08:42:35Z
Steps to reproduce 1. Create a Java 15 module project with module name MyMod. 2. In the project preferences of the project, enable project preferences for javadoc under Java Compiler > Javadoc preferences. 3. set "Malformed javadoc comments" to error 4. enable "Validate tag arguments " and save the settings. 5. Create Class Cls1 and paste the below code public class Cls1 { /** * @see MyMod/ abc */ public void main1() { } } as there is space between MyMod/ and abc an error is reported by Eclipse javadoc validator. that abc is not valid type. but the purpose above was to create a linked url with text abc which refers to the module MyMod. Expected behavior : This error should not be reported an upon hover on main1() function only abc link should be shown in javadoc of main1() and upon clicking it, the javadoc of Module myMod should be shown.
