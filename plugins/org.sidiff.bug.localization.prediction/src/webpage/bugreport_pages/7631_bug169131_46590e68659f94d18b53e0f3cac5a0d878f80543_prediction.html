Creation Time 2006-12-27T17:41:03Z
The Code Formatter currently has an option for "Blank lines within class declarations, Before first declaration" but it has no option for "After last declaration." Consider the following code: public class A { public void f() { } } The Code Formatter can add a blank line before the method, but it cannot add one after the method. This is the intended result: public class A { public void f() { } } This is the actual result: public class A { public void f() { } } This is an enhancement request to add a new option within the "Blank lines within class declarations" section for "After last declaration" or equivalently "Before closing brace."
