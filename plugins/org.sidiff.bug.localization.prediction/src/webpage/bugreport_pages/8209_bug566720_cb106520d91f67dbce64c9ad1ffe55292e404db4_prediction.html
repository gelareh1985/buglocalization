Creation Time 2020-09-07T08:27:55Z
In the below code public class X { public void main5() { public interface i{ } } public static void main(String[] args) { System.out.println(); } } the usage of keyword 'public' for interface i should give an error. Currently, an error is not reported for this.
Creation Time 2020-09-07T16:49:04Z
This is an issue as per sec 9.1.1 [http://cr.openjdk.java.net/~gbierman/jep384/jep384-20200506/specs/local-statics-jls.html#jls-9.1.1] The access modifier public (6.6) pertains only to top level interfaces (7.6) and member interfaces (8.5, 9.5), not to local interfaces (14.3).
