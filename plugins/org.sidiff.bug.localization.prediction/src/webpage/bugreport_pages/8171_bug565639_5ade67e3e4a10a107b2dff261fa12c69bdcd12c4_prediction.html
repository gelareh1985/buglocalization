In the example below, the stack trace should have at Main.main(Main.java:11) as the line where the exception occurred. It would also be possible to set a breakpoint at line 11. public class Main { static final String TEXT_BLOCK = """ 1 2 3 4 5 """; public static void main(String[] args) { throw new RuntimeException("This is line 11."); } } The stack trace, however, is: Exception in thread "main" java.lang.RuntimeException: This is line 11. at Main.main(Main.java:5) And setting a breakpoint at line 11 is not possible.

So for example, one can't put breakpoint in the example snippet at line 11 (inside main), because bytecode contains no line 11 at all.
