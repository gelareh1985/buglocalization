Creation Time 2019-05-14T15:01:39Z
In Java formatter, the `!` (not) operator is grouped under all the unary operators. So I can't specify to add a space after `not` operator only. IMHO the `not` operator is a particular unary operator, since is a logical operator. Usually you don't write `a&&b`, but `a && b`. And I find `! x` more readable that `!x`, while `+x` is IMHO more readable than `+ x`. So IMHO `not` operator should be handled specially. In other languages, the `not` operator is the word `not` itself. I think `not x` is better than `notx` (well, in this case it's also a compiler error...) So please, add an option to Java Formatter "White Space" section for `not` operator only.
