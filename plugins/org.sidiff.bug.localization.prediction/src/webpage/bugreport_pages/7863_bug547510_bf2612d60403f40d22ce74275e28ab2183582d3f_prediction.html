Now that JDT has integrated the chain completion feature from Code Recommenders, I would like to also integrate their subword completion: https://github.com/eclipse/recommenders/tree/master/plugins/org.eclipse.recommenders.subwords.rcp > JDT's Java Proposals require you to search for proposals by prefix match. > Subwords completion relaxes this constraint; you can search for proposals > using arbitrary subwords matches. E.g. content assist for "PlatformUI.getWorkbench().getManag|" offers getDecoratorManager, getIntoManager, etc. Nice side-effect: "camel case completion" also works case-insensitive ("workbench.gdm|" inserts getDecoratorManager()).
