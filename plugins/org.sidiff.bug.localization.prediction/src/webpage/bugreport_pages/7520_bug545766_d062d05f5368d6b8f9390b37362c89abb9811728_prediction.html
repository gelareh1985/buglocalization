Still some methods don't check for null before accessing #type: - checkIllegalNullAnnotation() - generateCode() - nameSourceStart(), nameSourceEnd() - Javadoc.resolveTypeReference() - probably OK because here we don't declare enums - QualifiedAllocationExpression: OK, not used for enum constants ProblemReporter.deprecatedMethod() has this nice block: if (allocationExpression.enumConstant != null) { start = allocationExpression.enumConstant.sourceStart; } start = allocationExpression.type.sourceStart; (almost nice, the last line should be in "else").
