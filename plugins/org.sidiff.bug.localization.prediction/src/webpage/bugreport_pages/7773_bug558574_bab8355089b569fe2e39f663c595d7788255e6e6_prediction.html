Creation Time 2019-12-22T21:13:57Z
Method like PrintWriter.printf() explicit document "@return This writer". This is relevant for analysis, so we don't bark at try (PrintWriter pw = ...) { pw.printf(...); } believing that printf created a new resource requiring closing. Even this should be understood: PrintWriter pw = new ... pw.printf(..).printf(..).close(); This becomes relevant with the fix for bug 463320.
