Creation Time 2020-06-24T06:41:03Z
``` public record Person(String name, int age){ } ``` When generating getters, the generated code is broken (inserted between `age` and the closing parenthesis) : ``` public record Person(String name, int age public String getName() { return name; } public int getAge() { return age; }) { } ``` This is happening because the position calculated in ASTRewriteAnalyzer.visit(RecordDeclaration node) is not correct.
